# 2D Fluid Simulator & Flipbook Generator

A web-based interactive fluid simulation tool designed for creating game assets. Simulate various liquids (water, blood, oil, slime), draw shapes, and export the results as seamless textures or animated flipbooks with depth maps.

![License](https://img.shields.io/badge/license-MIT-blue.svg)

## Features

- **Simulation Modes**:
  - **Wall Splatter**: Spray fluids onto a surface and watch them drip and streak.
  - **Floor Pool**: Simulate liquids pooling on a flat surface with organic spreading.
  - **One Click Pool**: Procedurally generate realistic pools (e.g., blood pools) with a single click.
- **Material Presets**: Built-in settings for Water, Blood, Oil, Honey, Slime, and Chocolate.
- **Export Tools**:
  - **Texture Export**: Save the current state as a high-res PNG (with optional Depth Map).
  - **Flipbook Export**: Record the entire simulation history and re-render it into a sprite sheet (flipbook) with frame blending and depth maps.
- **Touch Friendly**: Full support for mobile and touch devices.
